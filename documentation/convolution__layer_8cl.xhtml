This kernel performs a reshaping of the input tensor to a tensor used to perform convolution using GEMM.

Note
The data type must be passed at compile time using -DDATA_TYPE: e.g. -DDATA_TYPE=float
In case biases will be added to the convolution -DHAS_BIAS has to be passed to append the final matrix with 1 in each row.

This kernel reshapes the tensor's low three dimensions to single row for GEMM operation.

Note
Datatype should be given as a preprocessor argument using -DDATA_TYPE=type. e.g. -DDATA_TYPE=float
In case biases will be added in late stage, -DHAS_BIAS has to be passed to append the final matrix with 1 in each row.
