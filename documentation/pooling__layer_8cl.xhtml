Performs a pooling function of pool size equal to 2.

Note
Pooling stride must be passed using -DPOOL_STRIDE e.g -DPOOL_STRIDE=2. Supported strides are 1,2,3
Datatype must be passed using -DDATA_TYPE e.g. -DDATA_TYPE=float. Supported data types are F16, F32;
In case of average pooling -DPOOL_AVG must be provided otherwise max pooling will be performed.
Parameters
[in]input_ptrPointer to the source image. Supported data types: F16, F32
